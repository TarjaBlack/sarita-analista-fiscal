# Portfolio Sarita

Portfólio pessoal criado com Angular 22 e Tailwind CSS 4.

## Comandos principais

Instalar dependências:

```bash
npm install
```

Iniciar aplicação em desenvolvimento:

```bash
npm start
```

Build de produção:

```bash
npm run build
```

Executar testes unitários:

```bash
npm test
```

Executar testes com cobertura:

```bash
npm run test:coverage
```

## Configurações do projeto

- Angular `^22.0.0`
- Tailwind CSS `^4.3.2` usando `@tailwindcss/postcss`
- TypeScript `~6.0.2`
- Vitest para testes unitários
- `@angular/build` para build da aplicação

## Estrutura do projeto

- `src/` — código-fonte Angular
- `src/app/` — componente principal e lógica da aplicação
- `src/assets/` — imagens e recursos estáticos
- `public/` — arquivo `index.html` e metadados da aplicação
